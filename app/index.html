Course Pilot on CPS Interoperability Visual Documentation

This is a prototype of a D3 rendering of the curriculum of the course pilot to provide an easy to read visual documentation.
It is adapted by Lacramioara from Arnaud Lauret, the API Handyman.

Controls

Zoom in: Mouse scroll up
Zoom out: Mouse scroll down
Move: Mouse drag
Moving mouse over a node shows its documentation
Clicking on a node expand or collapse it
